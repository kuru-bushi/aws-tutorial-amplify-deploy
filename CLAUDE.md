# CLAUDE.md（プロジェクトインデックス）

このファイルは「どのドキュメントに何が書いてあるか」を示すインデックス兼、
ユーザーが**絶対に守ってほしいルール**だけを置く場所。詳細ルールは別ドキュメントに記載する。

## プロジェクトの目的

AWS Amplify Gen 2 の**チュートリアルを兼ねた学習用プロジェクト**。ユーザー自身が手を動かして学ぶことを重視する。

## 絶対に守るルール

- **Claude は `my-amplify-app/` のアプリソース（`src/` / `amplify/` / 設定ファイル）を編集しない。** 編集箇所とコードを提示し、ユーザーがコピーして手動編集する（詳細・例外は [docs/development-rules.md](docs/development-rules.md)）。
- **機密情報を成果物に残さない。** AWS アクセスキー / アカウント ID / ARN / パスワード / トークン等を、コード・ドキュメント・コミット・ログに記載しない（例として書く場合は `<your-account-id>` 等のプレースホルダを使う）。
